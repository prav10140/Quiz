# Python Console Quiz

A simple, customizable command-line quiz application. It loads questions from a JSON file, randomizes the order, and tracks the user's score.

## Features
- **Dynamic Content:** Loads questions from `questions.json`.
- **Randomization:** Questions appear in a different order every time.
- **Score Tracking:** Calculates final score and percentage.
- **Input Validation:** Prevents crashing on invalid user input.

## Setup & Run

1. **Prerequisites**
   - Python 3.x installed.

2. **File Structure**
   Ensure you have these two files in the same folder:
   - `quiz.py` (The application logic)
   - `questions.json` (The question database)

3. **Running the Quiz**
   Open your terminal/command prompt in the folder and run:
   ```bash
   python quiz.py
